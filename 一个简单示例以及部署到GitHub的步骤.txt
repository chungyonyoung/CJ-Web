步骤一：创建GitHub仓库
登录你的GitHub账号。
创建一个新的仓库，命名为 <你的用户名>.github.io（这个名字很重要，因为GitHub Pages会自动识别这个命名并部署到公网）。
初始化仓库时可以选择添加一个 README 文件。
步骤二：上传HTML文件
在你的计算机上，将之前创建的HTML文件保存为 index.html。
打开你新创建的GitHub仓库。
点击 Add file 按钮，然后选择 Upload files。
将 index.html 文件拖放到上传区域，或者使用文件选择器选择文件。
上传完成后，点击页面底部的 Commit changes 按钮。
步骤三：访问你的网页
上传完成后，GitHub Pages会自动将你的网页部署到公网。
你可以通过访问 https://<你的用户名>.github.io 查看你的网页。
示例
假设你的GitHub用户名是 exampleuser，你将能够通过 https://exampleuser.github.io 访问你的网页。
